## 解法 1: 双指针

时间复杂度: `O(n)`
空间复杂度: `O(1)`

```ts
function reverseList(head: ListNode | null): ListNode | null {
  let current = null
  let next = head
  while (next) {
    let temp = next.next
    next.next = current
    current = next
    next = temp
  }
  return current
}
```
