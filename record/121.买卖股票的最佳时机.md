## 解法 1: 动态规划

```ts
function maxProfit(prices: number[]): number {
  let [min, res] = [prices[0], 0]
  for (const price of prices) {
    res = Math.max(res, price - min)
    min = Math.min(min, price)
  }
  return res
}
```
