## 解法 1: 与下一个节点交换

> 想了一分钟,直接看题解 🙃,根本没思路的简单题,这题估计给我十分钟,也不一定想得出来

```ts
function deleteNode(root: ListNode | null): void {
  if (!root) return
  root.val = root?.next?.val!
  root.next = root?.next?.next!
}
```
