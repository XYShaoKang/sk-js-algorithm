## 解法 1: 贪心

从后往前贪心

```ts
function canJump(nums: number[]): boolean {
  let j = nums.length - 1
  for (let i = j - 1; i >= 0; i--) {
    if (nums[i] >= j - i) j = i
  }
  return j === 0
}
```
