## 解法 1: 累加求和

```ts
function missingNumber(nums: number[]): number {
  let [sum1, sum2] = [0, 0]
  for (let i = 0; i < nums.length; i++) {
    sum1 += nums[i]
    sum2 += i + 1
  }
  return sum2 - sum1
}
```

## Case

  { input: { nums: [0] }, output: 1 },
